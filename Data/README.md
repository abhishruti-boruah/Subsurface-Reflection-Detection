
# SHARAD Dataset

This project uses SHARAD radar observations collected from NASA's Planetary Data System (PDS).

Current Dataset

Region:
- Oxia Planum

Storage

The complete dataset is stored in Google Drive because GitHub is not intended for large scientific datasets.
